**Miniature Commission Building and Painting** - A Next.js app for my miniature building and painting business.

## Live Demo
foxxforge.com

## Tech Stack
- **Frontend:** Next.js (React), TypeScript, CSS (TailwindCSS)
- **Internationalization:** next-i18next
- **CI / Linter / Testing:** ESLint, Jest, React Testing Library
- **Deployment:** outsourced to Krimmel IT Service.

---

## Features

- Responsive gallery pages
- Internationalization in German and English
- SSR for Internationalization

## Getting Started

1.) Clone Repository:
```bash
git clone https://github.com/KatFoxx/foxxforge.git
cd foxxforge
```
2.) Install dependencies:
```bash
npm install
```
3.) Run development server:
```bash
npm run dev
```
4.) Open in browser:
http://localhost:3001

**Commands:**
npm run dev	| Start development mode
npm run build |	Build for production
npm run start	| Start production server
npm run test	| Run tests via Jest
npm run lint	| Run ESLint
